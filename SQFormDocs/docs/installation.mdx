### Install

```bash
$ npm install sqform
```

### Prerequisites

In order to use SQForm, its peer dependencies must be installed

```bash
$ npm install react react-dom @material-ui/core @material-ui/icons @material-ui/lab @material-ui/pickers yup
```

### Additional Setup

The `SQFormDatePicker` component relies on the Material-UI `LocalizationProvider` to work.

```jsx
import { LocalizationProvider } from '@material-ui/pickers';
import MomentAdapter from '@material-ui/pickers/adapter/moment';

<LocalizationProvider dateAdapter={MomentAdapter} locale={'en'}>
  <App />
</LocalizationProvider>;
```

:::tip

Visit the [SC+ Shared Components documentation](https://bitbucket.org/SelectQuote/scplus-shared-components/src/master/README.md) for setting up Material-UI
